# 분석 보고서 템플릿

## 목적

이 템플릿은 하나의 AI 코딩 하네스를 5개 비교 차원으로 분석하는 capstone 1단계 산출물입니다. 목표는 평가나 추천이 아니라, 설계 접근을 가치중립적으로 분해하고 근거 링크로 뒷받침하는 것입니다.

## 사용 방법

1. 대상 하네스를 하나 고릅니다.
2. [framework](../../framework.md)의 5개 차원을 기준으로 분석합니다.
3. 각 주장마다 기존 분석 문서, source permalink, 또는 비교 문서 링크를 붙입니다.
4. 마지막에 설계안으로 넘길 패턴과 tradeoff를 추출합니다.

## 근거 링크 요구

- 각 차원 행에는 최소 1개 이상의 기존 분석 또는 source 근거 링크가 있어야 합니다.
- `framework.md` 링크만으로는 충분하지 않습니다. 실제 하네스 또는 횡단 비교 근거를 함께 둡니다.
- 링크가 없는 판단은 “가정”으로 표시하고 설계안에서 다시 검증합니다.

## 체크리스트

- [ ] 5개 비교 차원을 모두 채웠다.

- [ ] 각 차원에 근거 링크가 있다.
- [ ] 추천·순위·별점 표현을 쓰지 않았다.
